Дивись, ідея у тому, щоб перебрати значення списку і порівнювати кожне із наступним. Якщо всі ці порівняння будуть коректними (кожна значення у списку строго менше за наступне) або список порожній, функція повертає True, а інакше False.
Які інструменти Python мені слід використати для втілення цієї ідеї у коді?
Я називаю цей крок в підказках "Загугли наступні терміни". Тобі знадобиться метод списків 'len()', оператор рівності '==0' для перевірки пустоти списку. Також, тобі знадобиться функція 'all()' для перевірки умови для всіх значень в списку. Всередині 'all()' створи новий список булевих значень з допомогою спискових виразів 'list comprehentions'. Щоб створити цей список використай 'range()' для проходження по вхідному списку і індексування списку 'list indexing' для порівняння пар значень. Використай оператор 'or' для об'єднання двох варіантів результату (для порожнього і не порожнього списку).
Я в цілому розумію, але все одно не виходить. Потрібно побачити остаточний код.
Ось він: 
def is_ascending(items: list[int]) -> bool:
    
    return len(items) == 0 or all([items[i] < items[i+1] for i in range(len(items)-1)])
Поглянь на параметр в діапазоні range. Я зменшив його на 1, інакше вираз items[i+1] хотітиме значення за межами списку. Щоб зробити код більш 'pythonic' (я раджу тоді загуглити також і цей термін), ми можемо використати оператор 'not' для перевірки пустоти списку і генератор замість спискового виразу. Оcь так: 
def is_ascending(items: list[int]) -> bool:
                    
    return not items or all(items[i] < items[i+1] for i in range(len(items)-1))
