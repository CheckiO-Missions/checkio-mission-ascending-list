In this particular solution I recommend you to use while statement: 
while (condition == True):
    # do something
But be carefull, if condition is always True, you will receive an infinite loop. So, inside while you should be able to make condition False or use break command. For example, the following code will execute N times: 
i = 0
while i < N:
    # do something
    i = i + 1 # better use increment 'i += 1'

As I said in the first hint, you must compare pair of values and check, if any of it is not strictly less then the next:

Ok, but this is only one comparison...how to compare values over the all list?
For this you need to combine both previous hints. As you start compare from the very beginning of list, you may use already declared variable 'i' as index to get the specific valuee from list: 
i = 0
while i < N:
    if items[i] >= items[i + 1]:
        # do something
    i += 1
As second index you use [i + 1] so must iterate not to the end of the list - len(items)), but only to len(items) - 1. Otherwise, i + 1 will be outside list and raise IndexError. Use it instead N: 
i = 0
while i < len(items) - 1:
    if items[i] >= items[i + 1]:
        # do something
    i += 1
Ok, but my solution still doesn't return any bool. What should I add?
Yes. If in comparison first value is not strictly smaller then the next, our list is not ascending. In this case, inside if, instead of 'do something', you need to return False. But if 'while' ends without this return, it means, list is ascending and after 'while' you must return True. Don't forget to declare function itself ;-) : 
def is_ascending(items: list[int]) -> bool:
    
    i = 0
    while i < len(items)-1:
        if items[i] >= items[i+1]:
            return False
        i += 1
    return True

I wonder if it is possible to write this solution in one line?
Yes. Using the generator inside all() and checking list emptiness with 'not items'. Pay attension, that using all() you should change comparison condition - all values must be strictly less then the next. Let's see this: 
def is_ascending(items: list[int]) -> bool:
                    
    return not items or all(items[i] < items[i+1] for i in range(len(items)-1))
