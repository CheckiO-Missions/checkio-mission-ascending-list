In this particular solution I recommend you to use while loop: 
while (condition == True):
    do something
But be carefull, if condition is always True, you will receive an infinite loop. So, inside loop you should be able to make condition False or use break command. For example, this code will execute 5 times: 
i = 0
while i < 5:
    do something
    i = i + 1 # better use increment 'i += 1'

def is_ascending(items: list[int]) -> bool: return not items or all(items[i] < items[i+1] for i in range(len(items)-1))
