I call this step of hints "Google the following terms". You will need 'len()' method of lists and equality operator '==0' to check if the list in empty. You also need 'all()' to check condition to all values in list. Inside 'all()' create list of bool using 'list comprehentions'. To create this list use 'range()' to iterate over the input list and 'list indexing' to compare pair of values. Use 'or' operator to join this two variants of result (for empty list and non empty).

Here it is: 
def is_ascending(items: list[int]) -> bool:
    
    return len(items) == 0 or all([items[i] < items[i+1] for i in range(len(items)-1)])
Look at the range parameter. I decrease it by 1, otherwise items[i+1] will be out of list. To make the code more 'pythonic' (I advise you to google this term as well), we may use 'not' operator to check list emptiness and generator instead of list comprehention. Like this: 
def is_ascending(items: list[int]) -> bool:
                    
    return not items or all(items[i] < items[i+1] for i in range(len(items)-1))
